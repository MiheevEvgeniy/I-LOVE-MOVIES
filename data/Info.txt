Данная программа предназначена для пользователей, которые когда-либо
сталкивались с проблемой хранения списка своих любимых книг, фильмов, сериалов или игр.
Приложение обеспечивает ввод/вывод и хранение выших данных в удобном для человека формате.
Сохранение данных производится автоматически, но при изменении данных в таблице, требуется сохранение.

Навигация по интерфейсу:
1. Область ввода находится в левом верхнем углу интерфейса и содержит 4 поля: название фильма,
оценка, статус завершенности и тип данных. Ниже данных полей находится кнопка ввода и галочка,
которую пользователь может поставить для включения функции проверки заполнения поля названия, либо
оставить элемент без галочки и работать без программной проверки поля;
2. Правее области ввода находится область поиска и фильтрации поиска данных. Поиск производится
по названию. Вывод результатов поиска производится в терминал справа от строки поиска;
3. В правом верхнем углу находится 2 области отвечающие за удаление данных:
        3.1. Верхняя областьпроизводит удаление ОТ выбранного номера
           элемента ДО другого выбранного номера элемента;
        3.2. Нижняя область производит удаление выборочно. Пользователь может
           составить список элементов, которые нужно удалить, а затем нажать
           на кнопку удаления, исключив выбранные данные из таблицы и хранилища данных
4. Таблица данных находится в нижней части экрана и содержит 5 столбцов: название, оценку,
статус, категорию, дату добавления элемента. Эта таблица является основным объектов в интерфейсе
и служит для наглядного и упорядоченного отображения данных пользователя;
5. Категория File панели управления программы находится слева от категории Info (кнопки вызова
данного окна) в левом верхнем углу приложения. В этой категории вы можете 3 функции: выйти
из приложения (может быть вызвано сочетанием клавиш ctrl + q ОСНОРОЖНО!
ЭТА ФУНКЦИЯ ЗАКРЫВАЕТ ПРОГРАММУ БЕЗ СОХРАНЕНИЯ), сохранить данные (ctrl + s),
открыть настройки (ctrl + e). Содержание настроек:
        5.1. Раздел визуальных настроек (слева):
                5.1.1. Функция смена языка. Переводит программу на один из двух языков;
                5.1.2. Изменение цвета элементов интерфейса. Для смены цвета в верхнем поле необходимо выбрать
                изменяемый элемент, после чего нажать кнопку "Сменить цвет" и выбрать желаемый цвет в
                палитре (при желании цвет можно сохранить в панели красок, нажав кнопку "Сохранить текущий цвет").
                После проведенных действий цвет элемента интерфейса будет изменен и окно выбора цвета
                можно будет закрыть. Для возвращения исходного стиля - нажмите на нижнюю кнопку
                категории смены цвета ("Тема по умолчанию").
        5.2. Раздел загрузки и выгрузки данных:
                5.2.1. Выгрузка в txt. Производит выгрузку в документ с расширением .txt в
                указанную пользователем папку.
                5.2.2. Выгрузка в excel. Производит выгрузку в документ с расширением .xls в
                указанную пользователем папку.
                5.2.3. Загрузка из txt. Производит загрузку из документа с расширением .txt в таблицу приложения.
                Требования к организации документа загрузки:
                    1) каждый загружаемый элемент должен быть на отдельной строке и
                    содержать 4 типа данных - категория, название, статус, оценка;
                    2) Формат строки должен иметь строго указанный вид:

                    Категория: Название(Статус Оценка/10)

                    ВАЖНО СТРОГОЕ СОБЛЮДЕНИЕ ПОСТАНОВКИ ПРОБЕЛОВ И СПЕЦИАЛЬНЫХ ЗНАКОВ!
                    ЛЮБОЕ ОТКЛОНЕНИЕ ОТ ОБРАЗЦА МОЖЕТ ПРИВЕСТИ К НЕКОРРЕКТНОЙ ЗАГРУЗКЕ!
                5.2.4. Загрузка из excel. Производит загрузку из документа с расширением .xls или .xlsx
                в таблицу приложения.

                    1) Первая строка должна быть заголовками типов данных - категория, название, статус, оценка.
